How it works

  1. Install Hermes Pong on your Mac (unzip → Applications).
  2. Link existing Hermes + Claude terminals (keeps Claude model & session), or start a New pair.
  3. When Hermes delegates, the task lands in the Claude Code window — you watch it work.

Requirements

macOS · Terminal.app · Hermes · Claude Code CLI · tmux
